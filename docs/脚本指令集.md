# StingerEngine 剧情脚本指令集

注意：

- 章节脚本文件放在 `StingerEngine_B/StingerEngine/chapters/`
- 每个章节必须暴露变量：`script_data = [...]`
- 启动章节时通过：`EngineClient.CreateGameUI("章节文件名(不含.py)")`

---

## 1. 脚本基本格式

```python
# -*- coding: utf-8 -*-

script_data = [
    {"type": "bg", "image": "textures/modTextures/demo/bg_cafe_rain"},
    {"type": "fade_in", "duration": 1.0},
    {"type": "text", "speaker": "旁白", "content": "故事开始了"},
    {"type": "return_to_title"}
]
```

说明：
- 每条命令是一个 `dict`
- 通过 `type` 字段分发到对应处理器
- 未识别 `type` 会记录警告并跳过

---

## 2. 执行与暂停机制（重要）

`GameUI.ExecuteUntilPause()` 会持续执行命令，直到出现“暂停点”。

常见暂停点：
- `text`：进入 `tap` 模式，等待玩家点击继续
- `menu`：进入 `menu` 模式，等待玩家选择
- `wait` / 带时长动画：进入 `wait` 模式，结束后自动继续
- 结尾：进入 `ended`

其他大多数命令（如 `bg`、`var`、`jump`）会立即执行并继续下一条。

---

## 3. 指令总览

### 3.1 流程与文本

#### `label`
定义标签（锚点），本身不暂停。

```python
{"type": "label", "name": "scene_start"}
```

参数：
- `name` (`str`): 标签名，供 `jump` 使用

---

#### `jump`
跳转到指定标签。

```python
{"type": "jump", "target": "scene_start"}
```

参数：
- `target` (`str`): 目标标签名

注意：
- 跳转时会清空内联命令队列（由 `condition` 注入的剩余命令）

---

#### `text`
显示对话文本，可选打字机速度。

```python
{"type": "text", "speaker": "苏瑶", "content": "欢迎光临。", "typewriter_speed": 0.03}
```

参数：
- `speaker` (`str`, 可选): 说话人名；空字符串会隐藏说话人面板
- `content` (`str`, 可选): 对话内容；非空时显示对话框
- `typewriter_speed` (`float`, 可选): 打字机每次更新间隔（秒）

行为：
- 自动进入 `tap` 暂停
- 若玩家点击时打字机还在播放，会先立即补完文本，再次点击才继续

---

#### `wait`
等待一段时间后继续。

```python
{"type": "wait", "duration": 0.5}
```

参数：
- `duration` (`float`): 秒数，`<= 0` 则不等待

---

#### `menu`
显示分支选项。

```python
{
    "type": "menu",
    "title": "你要怎么做？",
    "choices": [
        {"label": "route_a", "text": "选项A"},
        {"label": "route_b", "text": "选项B", "set_var": {"favorability": 1}}
    ]
}
```

参数：
- `title` (`str`, 可选): 菜单标题（显示在对话框）
- `choices` (`list[dict]`): 选项列表
  - `text` (`str`): 按钮文本
  - `label` (`str`, 可选): 选中后跳转到该标签
  - `set_var` (`dict`, 可选): 选中后批量写入变量（`self.ui.variables.update(set_var)`）

行为：
- 进入 `menu` 暂停，玩家选择后继续
- 没有可选项时会记录警告并自动继续

---

#### `condition`
条件分支执行，可内联命令列表。

```python
{
    "type": "condition",
    "condition": "favorability >= 1",
    "true_commands": [{"type": "jump", "target": "good_end"}],
    "false_commands": [{"type": "jump", "target": "normal_end"}]
}
```

参数：
- `condition` (`str`): 条件表达式
- `true_commands` (`list`, 可选): 条件为真时执行
- `false_commands` (`list`, 可选): 条件为假时执行

行为：
- 内联命令按顺序执行
- 如果中途遇到暂停命令，剩余内联命令会入队，恢复后继续

---

#### `var`
变量写入/运算。

```python
{"type": "var", "variable": "favorability", "operation": "add", "value": 2}
```

参数：
- `variable` (`str`): 变量名
- `operation` (`str`): `set | add | sub | mul | div`，默认 `set`
- `value` (`any`): 参与运算的值

值解析规则：
- `bool/int/float` 原样使用
- 字符串 `"true"/"false"` -> 布尔
- 字符串 `"none"/"null"` -> `None`
- 数字字符串 -> 数值
- 其他 -> 原字符串

注意：
- `div` 遇到除数 0 时结果置为 0
- 无效操作会抛 `ValueError`

---

### 3.2 画面与转场

#### `bg`
切换背景图。

```python
{"type": "bg", "image": "textures/modTextures/demo/bg_cafe_rain"}
```

参数：
- `image` (`str`): 背景贴图路径

---

#### `fade_in`
全屏淡入（黑 -> 透明）。

```python
{"type": "fade_in", "duration": 1.0}
```

参数：
- `duration` (`float`, 默认 `1.0`)

行为：
- 进入 `wait` 暂停，动画结束自动继续

---

#### `fade_out`
全屏淡出（透明 -> 黑）。

```python
{"type": "fade_out", "duration": 1.0}
```

参数：
- `duration` (`float`, 默认 `1.0`)

行为：
- 进入 `wait` 暂停，动画结束自动继续

---

#### `show_image`
显示 CG（支持双槽位交叉渐变）。

```python
{"type": "show_image", "image": "img_cg_good", "fade": 1.5}
```

参数：
- `image` (`str`): CG 贴图
- `fade` (`float`, 默认 `0`): 渐变时长

行为：
- `fade <= 0`：直接切换
- `fade > 0` 且已有旧图：交叉渐变并暂停等待
- `fade > 0` 且首张图：渐入并暂停等待

---

#### `hide_image`
隐藏当前 CG。

```python
{"type": "hide_image", "fade": 0.8}
```

参数：
- `fade` (`float`, 默认 `0`)

行为：
- `fade > 0` 时渐隐并暂停等待
- 完成后清空两个 CG 槽位

---

### 3.3 音频

#### `music`
控制背景音乐。

```python
{"type": "music", "file": "demo.bgm.relax", "action": "play"}
{"type": "music", "file": "demo.bgm.emotion", "action": "change", "fade": 2.0}
{"type": "music", "action": "stop", "fade": 1.5}
```

参数：
- `action` (`str`, 默认 `play`): `play | change | stop`
- `file` (`str`, `play/change` 时需要)
- `fade` (`float`, 默认 `0.0`): 切换/停止时淡出秒数

实现备注：
- 当前 BGM 通过 `PlayUISound(..., loop=True)` 实现
- `change` 会先停止当前音乐再播放新音乐

---

#### `sfx`
播放音效。

```python
{"type": "sfx", "file": "sfx_click", "loop": False}
```

参数：
- `file` (`str`): 音效名
- `loop` (`bool`, 默认 `False`)

---

### 3.4 角色立绘

支持通过 `CharacterManager` 动态创建角色控件（模板 `GameUI.character`）。

位置预设：
- `left`, `center_left`, `center`, `center_right`, `right`
- 也可直接传数值（`float`）作为 X 轴相对偏移

#### `character_enter`
角色入场（可淡入）。

```python
{"type": "character_enter", "id": "yao", "image": "char_yao_normal", "position": "left", "fade_in": 0.5}
```

参数：
- `id` (`str`): 角色唯一ID（必填）
- `image` (`str`, 可选): 立绘贴图
- `position` (`str|float`, 默认 `center`)
- `fade_in` (`float`, 默认 `0`)

---

#### `character_show`
直接显示角色（无入场动画）。

```python
{"type": "character_show", "id": "yao", "image": "char_yao_normal", "position": "center"}
```

---

#### `character_update`
更新角色图片/表情（可过渡）。

```python
{"type": "character_update", "id": "yao", "image": "char_yao_smile", "expression": "emo_happy", "transition": 0.3}
```

参数：
- `id` (`str`, 必填)
- `image` (`str`, 可选): 主立绘
- `expression` (`str`, 可选): 表情覆盖层贴图
- `transition` (`float`, 默认 `0`): `>0` 时执行“淡出->换图->淡入”

---

#### `character_play_anim`
播放角色动画。

```python
{
    "type": "character_play_anim",
    "id": "yao",
    "animdata": {
        "anim_type": "offset",
        "duration": 0.5,
        "from": [0, 0],
        "to": [10, 0]
    },
    "loop": False,
    "speed": 1.0
}
```

参数：
- `id` (`str`, 必填)
- `animdata` (`dict`, 必填): 动画定义
- `loop` (`bool`, 默认 `False`)
- `speed` (`float`, 默认 `1.0`): 若 `animdata` 含 `duration`，会按速度折算

行为：
- `loop=False`：暂停到动画结束
- `loop=True`：不暂停

---

#### `character_move`
移动角色到目标位置（smoothstep 补间）。

```python
{"type": "character_move", "id": "yao", "position": "right", "duration": 0.5}
```

参数：
- `id` (`str`, 必填)
- `pause` (`bool`，默认`False`)
- `position` (`str|float`, 默认 `center`)
- `duration` (`float`, 默认 `0.5`)

---

#### `character_scale`
缩放角色（相对默认尺寸倍率）。

```python
{"type": "character_scale", "id": "yao", "scale_x": 1.1, "scale_y": 1.1, "duration": 0.2}
```

参数：
- `id` (`str`, 必填)
- `scale_x` (`float`, 默认 `1.0`)
- `scale_y` (`float`, 默认 `1.0`)
- `duration` (`float`, 默认 `0`)

---

#### `character_hide`
隐藏并移除单个角色。

```python
{"type": "character_hide", "id": "yao", "fade_out": 0.3}
```

参数：
- `id` (`str`, 必填)
- `fade_out` (`float`, 默认 `0`)

---

#### `character_clear`
清除舞台上所有角色。

```python
{"type": "character_clear", "fade_out": 0.5}
```

参数：
- `fade_out` (`float`, 默认 `0`)

---

### 3.5 结束流程

#### `return_to_title`
返回主界面。

```python
{"type": "return_to_title"}
```

行为：
- 弹出当前剧情 UI
- 调用 `EngineClient.CreateMainInterfaceUI()`
- 标记流程结束

## 4. 条件表达式语法（`condition`）

支持：
- 比较运算：`> < >= <= == !=`
- 逻辑运算：`[and] [or] [not]`
- 圆括号：`(` `)`
- 变量名：字母/数字/下划线（不能数字开头）
- 字面量：数字、`True`、`False`、`None`

示例：

```python
"favorability >= 1"
"(score > 10) [and] (is_vip == True)"
"[not] (route_locked == True)"
```

注意：
- 逻辑关键字必须写成方括号形式，如 `[and]`
- 使用未定义变量会导致条件执行失败并按 `False` 处理

---

## 5. 常见问题

1. **Python 字面量大小写**
   - 脚本是 Python 文件，布尔值应写 `True/False`（不是 `true/false`）
2. **跳转标签必须存在**
   - `jump.target` 指向不存在标签会报错并停留当前流程
3. **用 demo 快速对照**
   - 可参考 `chapters/demo.py` 的多结局写法

## 6. 最小可运行示例

```python
# -*- coding: utf-8 -*-

script_data = [
    {"type": "var", "variable": "favorability", "operation": "set", "value": 0},
    {"type": "bg", "image": "textures/modTextures/demo/bg_cafe_rain"},
    {"type": "fade_in", "duration": 1.0},

    {"type": "text", "speaker": "旁白", "content": "这是一个最小剧本。"},
    {
        "type": "menu",
        "title": "请选择：",
        "choices": [
            {"label": "end_a", "text": "选项A", "set_var": {"favorability": 1}},
            {"label": "end_b", "text": "选项B"}
        ]
    },

    {"type": "label", "name": "end_a"},
    {"type": "text", "speaker": "系统", "content": "你选择了A"},
    {"type": "jump", "target": "finish"},

    {"type": "label", "name": "end_b"},
    {"type": "text", "speaker": "系统", "content": "你选择了B"},

    {"type": "label", "name": "finish"},
    {"type": "fade_out", "duration": 1.0},
    {"type": "return_to_title"}
]
```
